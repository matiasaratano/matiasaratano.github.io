About me

Hi! My name is Matias Aratano, I am 33 years old and I am from Argentina. I finished my career as a Systems Analyst at ORT Argentina. During my studies and training, I've acquired skills in React Native and Java as primary tools, along with QA tools such as Carina Test Automation Framework, Selenium, Appium, etc.

During the final project of my studies, we developed a mobile application "BDT Office" for a real client, using React Native for the front-end, Node.js for the back-end, developing the API and using MySQL as the database. To facilitate idea generation, we used Figma for brainstorming sessions. We also used Git and GitHub for version control, allowing seamless collaboration between team members as changes were implemented. I'm currently working on a web app, for a client who needs to organise registrations for different courses. I'm using React for the frontend and Firebase for the deploy. Firestore to store the registration data.
